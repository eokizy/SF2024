# SF2024
연암대학교 2024 스마트팜 프로젝트 깃허브입니다.


코드 관련 공지는 카카오톡과 리드미 파일을 통해 공지하겠습니다.


할 일은 develop branch의 README에서 확인 부탁드립니다.


# 마스터 코드 업데이트 방법


* 1. 완성된 코드만 업로드 가능합니다.


* 2. develop branch에서 완성된 코드를 보고 결정후 업데이트합니다.
